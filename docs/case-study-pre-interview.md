## 運用メモ（内部向け）

### 送付前に確認
- [ ] NDA や公開範囲の取り決め確認済み
- [ ] 顧客の担当者が「記事公開後の問い合わせ対応」を引き受けられる状態か
- [ ] 公開予定時期を合意しているか

### 事前シート回収後の準備
1. 数値データから **記事タイトル案を 3 つ** 作成（例: 「商談化率 15% → 42% を実現」系）
2. インタビューガイド（別 doc）を同社向けにカスタマイズ
3. 記事初稿のドラフト骨子を作っておく（ストーリー部分だけインタビューで埋める）
